== Install Haskell

=== Ubuntu

* Install Haskell
+
[source,shell]
----
sudo apt install -y --no-install-recommends ghc
----

* Upgrade Haskell
+
[source,shell]
----
sudo apt-get --only-upgrade install ghc
----

* Cabal
+
[source,shell]
----
cabal update
cabal install <package-name>
----
