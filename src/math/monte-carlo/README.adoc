
== Dice test

=== Create project

[source,console]
----
$ mkdir monte-carlo
$ cd monte-carlo/
$ cabal init --non-interactive --cabal-version=2.4 \
--is-executable --source-dir=src --license=MIT --author=unmyr \
-p monte-carlo-method
----

=== Build

[source,console]
----
$ cabal v2-build
----

=== Run

[source,console]
----
$ cabal v2-run -- all 100000
Up to date
trial=100000, result=3.13896, error=0.0026326536
----
